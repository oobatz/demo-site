# Промпт для создания лендинга мебельной студии

## Роль и Контекст
Ты — ведущий Senior Frontend-разработчик и UI/UX дизайнер, специализирующийся на премиальном E-commerce и интерьерах. Твоя задача — создать не просто сайт, а цифровое пространство, передающее атмосферу уюта, качества и стиля. Твой код должен быть безупречным, а дизайн — вдохновляющим на покупку.

## Задача
Создать полный код (HTML + CSS + JS) для современного одностраничного сайта (лендинга) мебельной студии под названием **"WoodArt Studio"**. Компания занимается изготовлением мебели на заказ и продажей готовых дизайнерских решений.

## Технические требования
1.  **Стек:** HTML5, Tailwind CSS (подключить через CDN), Vanilla JavaScript.
2.  **Структура:** Весь код в одном файле `index.html`.
3.  **Адаптивность:** Идеальное отображение на мобильных, планшетах и десктопах. Особое внимание уделить галереям на мобильных устройствах (свайпы).
4.  **Изображения:** Использовать качественные заглушки с `unsplash.source` по ключевым словам (interior, furniture, wood, cozy).
5.  **Шрифты:** Подключить Google Fonts. Для заголовков — элегантный шрифт с засечками (например, 'Playfair Display'), для текста — чистый гротеск (например, 'Lato' или 'Inter').

## Дизайн-система и Визуальный стиль
*   **Цветовая палитра:**
    *   Фон: Светлый, "воздушный" (белый `#ffffff`, мягкий бежевый `#f9f7f2`, светло-серый `#f3f4f6`).
    *   Текст: Темно-серый `#1f2937` (не чисто черный, чтобы смягчить контраст).
    *   Акцентный цвет: Природный зеленый `#059669` или терракотовый `#c2410c` (для кнопок и важных элементов).
    *   Вторичный: Цвет натурального дерева (коричневые оттенки).
*   **Стиль:** "Скандинавский минимализм" + "Премиум". Много воздуха (whitespace), крупные фотографии, тонкие линии, легкая анимация появления.
*   **Настроение:** Уют, надежность, экологичность, индивидуальность.

## Структура страницы (Секции)

1.  **Header (Шапка):**
    *   Логотип (текстовый, элегантный).
    *   Навигация (Каталог, На заказ, О студии, Контакты).
    *   Кнопка "Заказать звонок" (иконка телефона).
    *   Прозрачный фон, который становится белым при скролле.

2.  **Hero Section (Первый экран):**
    *   Большая атмосферная фотография интерьера на фоне (с затемнением или без, в зависимости от читаемости текста).
    *   Заголовок: "Мебель, которая создает дом".
    *   Подзаголовок: "Индивидуальное проектирование и изготовление мебели из массива дерева".
    *   Кнопка: "Рассчитать стоимость" (ведет на квиз/форму).
    *   Анимация: Плавное появление текста (fade-in-up).

3.  **Популярные коллекции (Catalog Preview):**
    *   Сетка (Grid) из 3-4 карточек товаров (Кухни, Шкафы, Столы, Декор).
    *   Карточка товара: Фото, Название, Цена, Кнопка "В корзину" или "Подробнее".
    *   Эффект при наведении: Увеличение фото (zoom), появление кнопки.

4.  **Процесс работы (Steps):**
    *   Горизонтальный таймлайн или сетка шагов: "Заявка -> Замер -> Эскиз -> Производство -> Доставка".
    *   Иконки в стиле line-art (тонкие линии).
    *   Краткое описание каждого этапа.

5.  **Квиз/Лид-магнит (Lead Generation):**
    *   Заголовок: "Узнайте стоимость вашей мебели за 1 минуту".
    *   Интерактивный элемент (простой JS): Выбор типа мебели -> Выбор материала -> Получение формы для контакта.
    *   Бонус за прохождение: "Скидка 10% на первый заказ".

6.  **Преимущества (Quality & Eco):**
    *   Акцент на материалах.
    *   Блок с фото текстуры дерева крупным планом.
    *   Текст: "Экологичные материалы", "Фурнитура Blum/Hettich", "Гарантия 5 лет".

7.  **Отзывы (Testimonials):**
    *   Слайдер или сетка с реальными фото интерьеров клиентов.
    *   Текст отзыва + Имя клиента.
    *   Дизайн карточек отзыва: легкий серый фон, скругленные углы.

8.  **Форма захвата (Contact/CTA):**
    *   Заголовок: "Приглашаем в шоу-рум".
    *   Поля: Имя, Телефон.
    *   Кнопка: "Записаться на просмотр".
    *   Адрес и часы работы рядом.

9.  **Footer (Подвал):**
    *   Минималистичный.
    *   Ссылки на соцсети (Instagram, Pinterest), Политика конфиденциальности.

## Требования к UX (Удобству)
*   **Визуальная иерархия:** Заголовки должны четко разделять секции.
*   **Микро-взаимодействия:** Плавное изменение цвета кнопок при наведении, плавный скролл по якорям.
*   **Доверие:** Реалистичные фото (заглушки должны выглядеть качественно), четкие цены или призыв к расчету.
*   **Скорость:** Оптимизированный код, отсутствие тяжелых библиотек.

## Инструкция для генерации
Напиши полный код, готовый к копированию и запуску в браузере. 
1.  Используй семантические теги HTML.
2.  Стилизуй через классы Tailwind CSS.
3.  Добавь JavaScript для мобильного меню, плавного скролла и простой логики квиза (переключение шагов).
4.  Код должен быть прокомментирован на русском языке.
5.  Убедись, что цветовая гамма передает ощущение тепла и уюта, в отличие от технических сайтов.
