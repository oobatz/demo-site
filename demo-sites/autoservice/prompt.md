# Промпт для создания лендинга автосервиса

## Роль и Контекст
Ты — ведущий Senior Frontend-разработчик и эксперт по UI/UX дизайну с 10-летним стажем. Твоя специализация — создание высококонверсионных лендингов для автомобильной тематики. Твой код чистый, семантический, а дизайн соответствует трендам 2024 года (минимализм, темная тема, неоновые акценты, глассморфизм).

## Задача
Создать полный код (HTML + CSS + JS) для современного одностраничного сайта (лендинга) автосервиса под названием **"AutoPro Elite"**.

## Технические требования
1.  **Стек:** HTML5, Tailwind CSS (подключить через CDN для быстрого рендеринга), Vanilla JavaScript.
2.  **Структура:** Весь код должен быть в одном файле `index.html` для удобства просмотра.
3.  **Адаптивность:** Полная поддержка мобильных устройств (Mobile First подход). Меню должно сворачиваться в "бургер" на экранах меньше 768px.
4.  **Производительность:** Использовать системные шрифты или оптимизированные Google Fonts (Inter/Roboto). Изображения должны иметь заглушки (placeholders) с тематикой авто.

## Дизайн-система и Визуальный стиль
*   **Цветовая палитра:**
    *   Фон: Глубокий темно-серый/черный (`#0f172a`, `#1e293b`).
    *   Текст: Белый и светло-серый для контраста.
    *   Акцентный цвет: Яркий "электрический" оранжевый (`#f97316`) или неоново-голубой (`#06b6d4`) для кнопок и важных элементов.
*   **Стиль:** "Технологичный премиум". Используй легкие градиенты, тени для создания глубины, скругленные углы карточек (rounded-xl).
*   **Типографика:** Крупные, жирные заголовки. Читабельный основной текст.

## Структура страницы (Секции)

1.  **Header (Шапка):**
    *   Логотип (текстовый "AutoPro").
    *   Навигация (Услуги, О нас, Цены, Контакты).
    *   Кнопка "Записаться" (Sticky/Fixed при скролле).
    *   Номер телефона крупно.

2.  **Hero Section (Первый экран):**
    *   Заголовок: "Профессиональный ремонт вашего авто с гарантией качества".
    *   Подзаголовок: "Диагностика, ТО и ремонт любой сложности за 60 минут".
    *   Две кнопки: "Записаться на сервис" (Акцентная) и "Наши услуги" (Контурная).
    *   Фоновое изображение: Затемненное фото современного гаража или процесса ремонта.

3.  **Преимущества (Why Us):**
    *   Сетка из 4-х карточек с иконками.
    *   Примеры: "Честные цены", "Оригинальные запчасти", "Комната ожидания с Wi-Fi", "Гарантия 1 год".
    *   Эффект при наведении (hover): легкое поднятие карточки и свечение.

4.  **Услуги (Services):**
    *   Grid-сетка с карточками популярных услуг (Замена масла, Ремонт ходовой, Диагностика двигателя, Шиномонтаж).
    *   В каждой карточке: Иконка/Фото, Название, Цена "от ... руб.", кнопка "Подробнее".

5.  **Блок доверия (Trust):**
    *   Счетчики: "5000+ довольных клиентов", "10 лет опыта".
    *   Логотипы брендов, с которыми работаем (серые, становятся цветными при наведении).

6.  **Форма записи (CTA Section):**
    *   Заголовок: "Запишитесь на бесплатную диагностику".
    *   Поля: Имя, Телефон, Марка авто, Выбор услуги (Select).
    *   Кнопка отправки с анимацией загрузки.

7.  **Footer (Подвал):**
    *   Контакты, адрес с картой (заглушка), ссылки на соцсети, копирайт.

## Требования к UX (Удобству)
*   **Ясность:** Кнопки призыва к действию (CTA) должны быть визуально выделены.
*   **Навигация:** Плавный скролл (smooth scroll) при клике на пункты меню.
*   **Интерактивность:** Добавь микро-анимации (появление элементов при скролле, ховер-эффекты на кнопках).
*   **Форма:** Валидация полей (чтобы нельзя было отправить пустую форму).

## Инструкция для генерации
Напиши полный код, готовый к копированию и запуску в браузере. Не используй внешние локальные файлы (картинки бери с `unsplash.source` или аналогичных сервисов, стили через CDN). Код должен быть прокомментирован на русском языке в ключевых местах.
